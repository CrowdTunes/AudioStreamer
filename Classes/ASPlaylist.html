ASPlaylist Class Reference

Inherits from NSObject
Declared in ASPlaylist.h

Overview

The ASPlaylist class is intended to be a wrapper around the AudioStreamer class for a more robust interface if one is desired. It also manages a queue of songs to play and automatically switches from one song to the next when playback finishes.
